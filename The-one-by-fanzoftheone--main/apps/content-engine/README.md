# Content Engine
**Automated AI Video & Content Pipeline**

One prompt in. Finished content out. The engine handles scripting, visuals, voice, and video assembly autonomously.

## Pipeline
1. **Script** — LLM (Claude/DeepSeek/Grok) writes the content
2. **Visuals** — Leonardo.ai generates on-brand imagery
3. **Voice** — ElevenLabs synthesizes the narration
4. **Assembly** — FFmpeg renders the final video
