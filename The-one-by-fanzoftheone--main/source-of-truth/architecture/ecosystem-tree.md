# THE ONE — Ecosystem Tree

## Purpose
The One is a dual-engine ecosystem.

Revenue engine builds and funds life.
Growth engine helps the human think better, grow better, and decide better.

## Core split

THE ONE
├── Revenue Engine
│   ├── Builder Studio
│   ├── Modules
│   ├── Marketplace
│   ├── Apps
│   ├── Service SaaS
│   └── Agent Tools
│
└── Growth Engine
    ├── Companion
    ├── Memory
    ├── Reflection
    ├── Planning
    ├── Pattern Detection
    └── Life Direction

## No-drift rules

Labs test ideas.
Modules stabilize ideas.
Apps monetize stable systems.
Marketplace distributes stable systems.
Warehouse stores memory and receipts.
Companion remains a growth partner.
User remains orchestrator.
