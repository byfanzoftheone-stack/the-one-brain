# THE ONE — Promotion Pipeline

## Idea Flow

Every idea moves through this pipeline.

idea
↓
lab test
↓
validated pattern
↓
module
↓
app
↓
marketplace


## Stage 1 — Idea

Ideas come from:

user
labs
agents
companion insights

Ideas do not become products yet.


## Stage 2 — Lab

Labs test ideas.

Labs answer:

Does this work?
Does this solve a real problem?
Can it repeat?


## Stage 3 — Validated Pattern

When a lab proves useful repeatedly, it becomes a pattern.

Patterns move to module creation.


## Stage 4 — Module

Modules are installable building blocks.

A module must include:

module.json
backend service
frontend UI (optional)
clear inputs/outputs


## Stage 5 — App

Apps combine multiple modules into a working product.

Examples:

CRM
Construction Ops
Pool Management
Content systems


## Stage 6 — Marketplace

Marketplace distributes apps and modules.

Marketplace items must have:

name
description
pricing
installation package


## Promotion Rule

Nothing skips stages.

Idea cannot jump directly to app.
