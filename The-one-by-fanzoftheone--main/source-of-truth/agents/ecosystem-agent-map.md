# THE ONE — Ecosystem Agent Map

## Core rule

The ecosystem does not use random agents.

It uses:

- gate agents
- worker agents
- warehouse memory
- companion support
- user orchestration

User remains the orchestrator.

Each ecosystem zone has:
- one gate agent
- supporting worker agents
- memory feedback into warehouse

---

## Colony model

The ecosystem works like a colony:

User = orchestrator

Each zone = one gate agent
Each gate agent = several worker agents
Workers report to warehouse
Warehouse stores patterns
Builder uses learned patterns
Marketplace sells built products
Companion supports the human behind the whole system
