**by fanzoftheone — Because you are The One**

## Core Crew
- Planner
- Worker
- Auditor
- Coordinator
- Warehouse

## What each part does
### Planner
Breaks a user goal into executable steps.

### Worker
Runs tasks through the tool layer.

### Auditor
Checks results for missing steps, null output, and error output.

### Coordinator
Connects planner, worker, auditor, and warehouse into one execution flow.

### Warehouse
Stores task history and results in persistent memory.
