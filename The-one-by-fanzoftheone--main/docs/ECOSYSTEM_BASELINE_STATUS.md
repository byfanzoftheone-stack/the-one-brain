# THE ONE Ecosystem Baseline Status

This archive includes a low-cost, provider-routed baseline with:

- runtime orchestrator and provider router
- OpenAI / Claude / Ollama bridge stubs
- tool registry and operational tools
- opportunity engine and ranker
- builder / auditor / launcher agent roles
- worker loop for opportunity → build → audit → marketplace sync
- marketplace publish/install/status/sync engines
- scanner layer for industry, niche, and automation signals
- recipes for build, deploy, and marketplace actions
- warehouse receipts and learning folders

## Important

This package is designed as a **working baseline**, not a fully production-hardened SaaS operating system.
To make it fully live you still need:

- valid API keys for hosted providers
- backend route wiring from frontend actions
- deployment credentials for GitHub / Railway / Vercel
- durable database integration if you want persistent multi-user state
